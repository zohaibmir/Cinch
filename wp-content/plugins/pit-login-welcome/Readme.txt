== Description ==
### PiT Login Welcome by http://www.webingnet.com

This is a simple plugin. Pit Login Welcome is a Plugin which can give a Random Welcome message to your blog/Website's users when login.


== Installation ==

1. Search 'Pit-Login-Welcome' to Plugins library.
2. Install the plugin by click the Install option.
3. Activate the plugin.
4. That's all. Its working automatically. OR

1. Download the 'Pit-Login-Welcome.zip' file from WordPress plugins directory.
2. Upload to '/wp-content/plugins/' directory
3. Unzip the plugin folder
4. Activate the plugin through the 'Plugins' menu in WordPress Dashboard.
5. That's all. Its working automatically.
